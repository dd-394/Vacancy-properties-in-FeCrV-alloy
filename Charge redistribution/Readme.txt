The raw data of relationship between charge redistribution and atomic movements are recorded in Charge redistribution.xlsx. 

Due to their large size (142 MB per file × 128 files ≈ 18 GB), the CHGCAR files containing charge density data are not included in this repository. To reproduce the charge density analysis, we provide all necessary OUTCAR, POSCAR and the Python scripts used for post-processing. The CHGCAR files can be generated by performing VASP calculations using these inputs. If you require access to the raw CHGCAR files for any reason, please contact us and we will be happy to arrange a transfer.
